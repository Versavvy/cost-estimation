Homework Help generates one diagram image per session, regardless of walkthrough depth or whether the upload contains a figure. This slider is kept for reference only.

3 Walkthrough depth

The build pipeline is 5 LLM calls (steps A · parallel B · demo C · 3A explanation · answer key D). A→B→C can regenerate when the self-check fails — the retry factor scales those three.

4 Interaction

After the demo the child works their own question: each typed attempt is validated by one LLM call, and each mid-step follow-up question (“ask”) is another.

5 Audio (Inworld TTS-2)

Charged per character. Audio narrates the 3A explanation, every solution & demo step, the intros / completion, and each piece of feedback / Q&A reply. Audio is costed on the measured average narration length — measured across real homework sessions — and is route-specific: calculation (o3) sessions narrate far less than prose (gpt-4.1) explanation sessions. Audio is costed on the measured average lesson narration — all three mastery levels combined (~33 narration chunks + ~18 worked-example steps + the intro greeting) — at Inworld's $17 / 1M-character Developer plan, giving ≈ $0.48/lesson of audio (down from the old $1.50–$2.00 line). Defaults verified against live published pricing (June 2026): o3 & gpt-4.1 both $2 input / $8 output per 1M tokens; Gemini 3.1 Flash Image (Nano-Banana-2) $0.10 per image (conservative round-up from the ~$0.067 standard-tier list); Inworld TTS-2 $17 per 1M characters (Developer plan) — dropping further at higher commit tiers and to ~$10 at enterprise scale. omni-moderation is free. Cached input on the OpenAI models is ~$0.50/1M; the Batch API is ~50% off. If you have negotiated or volume rates, type them in — these are list prices, not your invoice.
